# Rishi's Lily Farm and Exotic Plants

A collaborative web development project for an exotic plant nursery and farm.

## Project Overview

This is a modern, responsive website for Rishi's Lily Farm and Exotic Plants, featuring:
- Product catalog and shopping cart
- Customer gallery and reviews
- FAQ section for plant care
- Contact information and location
- Admin panel for content management
- Firebase integration for database and authentication

## Technology Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** Firebase (Firestore, Authentication, Hosting)
- **Development:** Live Server for local development
- **Version Control:** Git with GitHub

## Quick Start

1. **Clone the repository:**
   ```bash
   git clone [repository-url]
   cd rishis-lily-farm
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000`
